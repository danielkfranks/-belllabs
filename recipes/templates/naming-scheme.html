Naming Scheme

Guidelines for naming files and directories in our project.

Home | Gallery | Get Involved | Join the Community

File Naming Guidelines

  • Use lowercase letters and hyphens (-) to separate words (e.g., network-operations.html).
  • Keep file names descriptive but concise.
  • Use the appropriate file extension (.html, .css, .js, etc.).

Directory Structure

  • Place all HTML files in the root or relevant subdirectories (e.g., recipes, templates).
  • Keep assets like images, stylesheets, and scripts in dedicated directories (e.g., img, css, js).
