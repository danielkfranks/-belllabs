How to Contribute

We welcome contributions from everyone! Follow the steps below to get started:

  1. Clone our repository from GitHub.
  2. Use the templates provided in the templates directory for creating new pages.
  3. Follow the naming scheme outlined in the Naming Scheme.
  4. Submit a pull request with your changes.
